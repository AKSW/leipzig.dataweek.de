Scads Tutorials

ScaDS.AI Trainings

As part of Data Week, ScaDS.AI Dresden/Leipzig is offering three days of Data Science training for interested scientists, students, and especially non-scientific users. Within teaching and training, both technical and methodological priorities are set. The technical focus includes tools for data analysis, distributed computing, and machine learning such as R and Python. On the other hand, the use of tools with the help of containers and virtual machines plays a role. The methodological focus includes instruction in methods of data analysis, statistics, and machine learning. The following trainings are offered as part of the Data Week Leipzig 2023:

  • Introduction to data analysis with the Python programming language - getting to know the tools Jupyter Notebook, Numpy and Pandas and data preprocessing (4 hours)
  • Introduction to machine learning / AI with the Python programming language (4 hours)
  • Visualization of data with the programming language R (4 hours)

The trainings will take place on 26.06, 28.06 and 29.06. The allocation of the trainings according to individual days will be coordinated in the course of the progressing program planning. The number of participants is limited to a maximum of 20 persons per training. Registration takes place when purchasing Data Week Leipzig tickets via the ticketing system. ScaDS.AI Schulungen
